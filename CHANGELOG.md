# Changelog

All notable changes to mojo-ui will be documented in this file.

## [0.1.0] - Unreleased

### Added
- 28 React components (buttons, forms, layout, navigation, feedback, overlay, data display)
- Tailwind CSS preset with nature-themed color tokens (moss, bark, coal, clay, river, paper, sun, ember)
- Glassmorphism shared style utilities
- Font families: Tiny5 (display), KyivType Sans (UI), KyivType Serif (body), JetBrains Mono (code)
- Interactive showcase for component previews
- Design tokens reference page
- Component API documentation
- ARIA accessibility support
- TypeScript declarations
- ESM and CJS build outputs
